🏗️ Setup Instructions

Clone the repo:

git clone https://github.com/Saikrishna1124/flask-portfolio-website.git
cd flask-portfolio-website


Install dependencies:

pip install -r requirements.txt


Run the Flask app:

python app.py


Open in browser:

http://127.0.0.1:5000/
